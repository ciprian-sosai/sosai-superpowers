# sosai-superpowers — Agent Guidelines

This file contains the same guidelines as CLAUDE.md, applied to all agent types.

See [CLAUDE.md](CLAUDE.md) for the full contributor guidelines.

Key points for agents:
- Read existing skills before proposing changes — understand the cowork-first philosophy
- Skills here are for business professionals, not developers — no technical jargon
- Every PR needs a real problem statement and human approval before submission
- Domain-specific skills belong in separate plugins, not here
- Identify your model and harness in every PR
